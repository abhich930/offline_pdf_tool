# Roadmap

## V1 (current)
- Merge/split/reorder/rotate/delete/compress flow skeleton.
- HTML->PDF, Markdown->PDF, JSON->PDF, images<->PDF converters.
- One-task-at-a-time execution model.

## V1.1
- Replace placeholder conversion logic with richer, validated backends.
- Add drag/drop and share-sheet platform integrations.
- Add encrypted workspace toggle and app lock on mobile.

## V2
- Evaluate open-source office format conversion quality.
- Add optional best-effort DOC/DOCX/XLSX conversion if quality acceptable.

## TODO (Before Store Release)
