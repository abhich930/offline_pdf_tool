# Offline PDF Tool

This is a sample Markdown file for testing `.md` to `.pdf` conversion.

## Supported formatting

- Headings
- Paragraphs
- Bullet lists
- Numbered lists
- Block quotes
- Code blocks

> Everything stays local on the device.

### Example code

```dart
void main() {
  print('Markdown to PDF');
}
```

1. Pick a Markdown file
2. Convert it offline
3. Save the generated PDF
